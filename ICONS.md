# إنشاء الأيقونات للتطبيق

لإكمال PWA، تحتاج إلى إضافة أيقونات للتطبيق.

## المتطلبات

- `icon-192.png` - حجم 192x192 بكسل
- `icon-512.png` - حجم 512x512 بكسل

## طرق إنشاء الأيقونات

### الطريقة 1: استخدام أداة عبر الإنترنت
1. اذهب إلى [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
2. أو استخدم [RealFaviconGenerator](https://realfavicongenerator.net/)
3. ارفع صورة واختر حجم 192x192 و 512x512

### الطريقة 2: استخدام برنامج تصميم
- استخدم Photoshop, GIMP, أو أي برنامج تصميم
- أنشئ صورة مربعة 512x512 بكسل
- احفظ نسخة 192x192 و 512x512

### الطريقة 3: استخدام أيقونة بسيطة
يمكنك استخدام أيقونة بسيطة بلون واحد أو نص:
- لون خلفية: #2196F3 (أزرق)
- نص: "ح" أو "م" (للمحاسبة)
- خط: Arial أو خط عربي واضح

## وضع الملفات

ضع الملفات في المجلد الرئيسي:
```
Source/
├── icon-192.png
└── icon-512.png
```

## ملاحظة

التطبيق سيعمل بدون الأيقونات، لكنها مطلوبة لتثبيت PWA بشكل صحيح.
