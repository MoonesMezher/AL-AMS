# دليل البدء السريع

## الخطوة 1: تثبيت المتطلبات

```bash
npm install
```

## الخطوة 2: تشغيل الخادم

```bash
npm start
```

سيعمل الخادم على: `http://localhost:3000`

## الخطوة 3: فتح التطبيق

افتح المتصفح واذهب إلى: `http://localhost:3000`

## الخطوة 4: تسجيل الدخول

- **البريد الإلكتروني**: `admin@example.com`
- **كلمة المرور**: `admin123`

**ملاحظة مهمة**: 
- بعد تسجيل الدخول، ستبقى مسجلاً لمدة 30 يوم
- لن تحتاج لتسجيل الدخول في كل مرة تفتح فيها التطبيق
- جميع البيانات متزامنة عبر جميع الأجهزة

## المزامنة عبر الأجهزة

1. سجّل الدخول من أي جهاز باستخدام نفس البيانات
2. جميع التغييرات ستظهر على جميع الأجهزة تلقائياً
3. يعمل التطبيق بدون إنترنت ويتم المزامنة عند عودة الاتصال

## استكشاف الأخطاء

### الخادم لا يعمل
- تأكد من تثبيت المتطلبات: `npm install`
- تأكد من أن المنفذ 3000 غير مستخدم
- تحقق من وجود Node.js مثبت

### البيانات لا تتزامن
- تأكد من أن الخادم يعمل
- تحقق من الاتصال بالإنترنت
- افتح Console المتصفح للتحقق من الأخطاء

### لا يمكن تسجيل الدخول
- تأكد من أن الخادم يعمل
- استخدم البيانات الافتراضية: `admin@example.com` / `admin123`
- تحقق من Console المتصفح للأخطاء
